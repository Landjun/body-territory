# 轻松跑

> 模块:无伤跑法
> 状态:🔲 未学 / 📖 学习中 / ✅ 能讲清楚
> 个人学习笔记,非医疗建议。

## 一句话讲清(费曼核心)

轻松跑是低强度、能边跑边说话、跑完不瘫的跑——它是整个训练体系的"地基量",负责在不制造大疲劳的前提下,默默积累有氧能力和组织耐受。

## 费曼检验问题
- 轻松跑要"轻松"到什么程度才算跑对了?(怎么自测强度)
- 为什么训练里大部分的量都应该是轻松跑?
- 轻松跑跑得慢,是不是就等于白练了?

## 详细笔记

**它在练什么**
低强度有氧持续刺激,促进心血管、有氧代谢能力和肌腱组织的适应,而冲击和疲劳代价都较低。它是"可大量、可持续"的训练。

**怎么自测够不够轻松**
- **谈话测试**:能完整、轻松说出一句话不明显喘;
- **跑后**:不瘫,能继续工作学习;
- **次日**:无明显加重的疲劳或疼痛。
满足这三条,慢一点完全没关系。

**为什么大部分量应是轻松跑**
高强度效果好但**恢复成本高**,做多了滑向受伤;轻松跑成本低、可近乎天天进行,长期累积出厚实的有氧底子,反而抬高了上限。所以成熟的训练结构里,低强度占大头、高强度只占小比例。

**对我(恢复期)的意义**
当前以安全恢复为主,轻松跑几乎就是主菜:先把"能稳定、不受伤地跑"做扎实。详见 docs 版[轻松跑原则](../../docs/02-running-safety/easy-run-principles.md)。

## 类比与图示

像**往银行定投**:每次不多、风险低,但长期复利出厚实本金。指望靠几次高强度"一夜暴富",反而容易爆仓(受伤)。

## 常见误区
- "跑这么慢=没练"——轻松跑是基础,基础越厚上限越高。
- 每次都想冲——把轻松跑跑成中强度,恢复跟不上。
- 用配速绑架——以"能说话/次日不加重"为准,而非某个配速。
