# 不卖惨、不鸡血:安全跑者的写作口径

> 一句话讲清:可信来自真实和克制——不神化苦难、不夸大效果、不炫耀成绩、不装专家,只讲真实训练、真实复盘、真实方法、真实边界。

## 这篇解决什么问题

它是一份"写作护栏",防止内容滑向自我感动、贩卖焦虑或伪专业。守住它,内容才能长期立得住。

## 七条护栏

1. **不卖惨**:可以写挫折和教训,但目的是复盘方法,不是博同情。
2. **不神化苦难**:"带伤硬扛"不是荣誉,降级才是纪律。
3. **不夸大效果**:不写"练了 X 就好了""X 天恢复"这类因果断言。
4. **不炫耀成绩**:不靠配速/跑量定义价值,也不晒来立人设。
5. **不做医疗建议**:不诊断、不开方,涉及伤病一律带免责并建议转介。
6. **不装专家**:学习者口径,讲"我的理解和实践",承认卡点和不确定。
7. **不强行升华**:不是每篇都要扯到网络安全或人生意义。

## 反例 → 正例

| 别这么写(✗) | 这么写(✓) |
|---|---|
| "忍着剧痛跑完,这就是意志!" | "疼到 4 分,我改成了快走+力量,并复盘了诱因。" |
| "这套动作三天治好我的膝盖" | "这个动作针对臀中肌薄弱,我在无痛范围内练,效果待长期验证。" |
| "破 PB!配速碾压!" | "今天安全完成,次日无加重——对我这就是胜利。" |
| "作为半个康复师我判断这是……" | "这是个人复盘,不是诊断;出现红旗信号我会去看医生。" |

## 和安全跑者的关系

这正是[宣言](../01-identity/safe-runner-manifesto.md)里"不靠卖惨和鸡血、不崇拜硬撑"的落地口径。身份是什么,内容口径就该是什么。

## 和网络安全纪律的关系

像写安全披露:**不夸大危害骗眼球,不隐瞒边界装权威**。专业感来自分寸,不来自音量。

## 常见误区

- 以为"真实=暴露一切惨"——真实是诚实+克制,不是卖惨;
- 以为"克制=没观点"——可以有强观点,但建立在事实和方法上;
- 为流量临时破例一次——信任是一次性资产,破一次就难修。

## 今天可以怎么用

回看你最近想发的一段话,用上面"反例→正例"自检一遍,把任何卖惨/夸大/装专家的措辞改掉。
