# 安全跑者金句

可复用的核心句子，用于内容、讲稿、自我提醒。它们都来自这个仓库的精神内核。

```markdown
网络安全是我要成为谁，跑步是我用什么身体去成为那个人。
```

```markdown
一个用身体的纪律映照安全的纪律的人。
```

```markdown
降级不是失败，是纪律。
```

```markdown
安全完成就是胜利。
```

```markdown
力量重建不是跑步的附属品，而是安全跑步的地基。
```

```markdown
训练不是征服身体，而是理解身体边界。
```

```markdown
不受伤地持续，才是真正长期主义。
```

```markdown
没有记录，训练就很难变成系统。
```
