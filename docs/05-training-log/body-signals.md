# 身体信号库

> ⚠️ 个人训练记录,非医疗建议;出现持续/加重疼痛、肿胀、麻木、步态改变等,请咨询专业人士。

## 本页目的

把常见身体信号集中起来，并说明它们如何影响训练决策。信号是输入，决策是输出。

## 信号与决策

| 信号 | 可能含义 | 对训练决策的影响 |
|---|---|---|
| 疼痛 | 局部负荷过大或损伤 | 按疼痛分级判断能否跑，3+ 不跑 |
| 紧张 | 负荷/恢复/力量不足 | 改拉伸+轻力量，回看跑量 |
| 疲劳 | 恢复不足 | 降级，优先睡眠 |
| 睡眠差 | 恢复资源不足 | 降一级，不做高强度 |
| 心情差 | 压力大、动力低 | 选 E 级保底，不硬撑 |
| 训练后兴奋 | 强度合适、状态好 | 维持，不盲目加量 |
| 训练后萎靡 | 负荷偏大 | 下次降级 |
| 次日加重 | 上次负荷过大 | 降级并复盘 |
| 步态变化 | 高风险信号 | 停止跑步，优先处理 |

## 核心原则

- 信号优先于计划；
- 多个负面信号叠加时，叠加降级。

## 最小行动

记录今天最明显的一个身体信号，并写下它让你做了什么决定。
