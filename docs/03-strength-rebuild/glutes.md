# 臀部

> 个人训练学习笔记，非医疗建议。动作出现疼痛请停止并参考[红旗信号](../02-running-safety/red-flags.md)。

## 本页目的

讲清楚臀部在跑步里到底干什么、为什么"练臀"几乎是无伤跑步的第一性问题，以及具体怎么练、怎么自测。

## 这个部位在跑步里干什么

臀部不是一块肌肉，而是一组分工不同的肌肉：

- **臀大肌**：最大、最有力的伸髋肌。跑步里负责把身体"向前推"，是推进力的主要来源之一；落地时也帮忙缓冲(离心控制髋屈)。
- **臀中肌 / 臀小肌**：位于髋的外侧，负责**髋外展**，更关键的是在**单腿支撑**时把骨盆稳住。跑步本质上是不断的单腿支撑，这两块肌肉每一步都在防止骨盆向对侧塌下去。

## 为什么对无伤跑步这么重要

跑步是"单腿交替支撑"的运动：任何时刻要么单脚撑地，要么腾空。每一次落地，身体重量压在一条腿上，骨盆需要被稳稳托住。

- 臀中肌不够强 → 支撑时**对侧骨盆下沉**、躯干歪向支撑腿(代偿性摇摆)，连带髋内收、**膝内扣(knee valgus)**。
- 膝内扣 + 骨盆不稳 → 常和膝前痛、髂胫束摩擦、髌股关节压力增大等问题相关。
- 臀大肌偷懒 → 推进改由腘绳肌、小腿、腰背"接管"，这些部位被迫超负荷，容易紧张和劳损。

所以很多看起来是"膝盖问题""小腿问题"的麻烦，根子常常在臀。**练臀 = 给膝、踝、腰减负**。

## 代表动作

| 动作 | 主练 | 难度 |
|---|---|---|
| 臀桥 | 臀大肌 | 入门 |
| 单腿臀桥 | 臀大肌 + 骨盆稳定 | 进阶 |
| 蚌式开合 | 臀中肌 | 入门 |
| 侧卧髋外展 | 臀中肌 | 入门-进阶 |
| 单腿站立 | 臀中肌(稳定) | 入门 |
| 弹力带怪兽走 | 臀中肌(动态) | 进阶 |

## 动作详解

### 臀桥
- **起始**：仰卧屈膝，双脚与髋同宽，脚跟离臀部一掌距离。
- **过程**：先把骨盆轻微后倾(腰贴地)，再用**臀部**把髋抬到肩-髋-膝大致成直线，顶端**夹紧臀**停一下，缓慢落下。
- **发力感**：应该是臀在烧，不是腰或腿后。
- **退阶**：缩小幅度、不停顿。**进阶**：单腿臀桥、顶端停留更久。

### 蚌式开合
- **起始**：侧卧，双膝屈约 90°，脚跟相触，骨盆垂直地面(别后仰)。
- **过程**：脚跟保持相触，用臀外侧把上方膝盖打开，控制着合回。
- **发力感**：髋外侧(裤兜位置)酸。
- **常见偷动作**：骨盆向后仰用惯性翻——保持骨盆稳定才算数。

### 单腿站立(臀中肌稳定自测兼训练)
- **过程**：单脚站立，另一腿微抬，保持**骨盆水平**不向抬腿侧掉。
- **进阶**：闭眼、站软垫、加上半身轻微动作干扰。

## 常见错误

- **用腰顶而不是用臀**：臀桥顶端腰发酸 → 先做骨盆后倾、收紧核心，降低幅度找臀的感觉。
- **蚌式靠骨盆翻滚偷动作**：把动作做小、骨盆锁住。
- **只追求次数**：失去顶端收紧感的快速重复价值很低，宁可慢而稳。
- **膝内扣不自知**：对镜子或录像看单腿动作时膝是否往内倒。

## 怎么练进训练里

- 臀属于"高频友好"部位，适合作为[5 分钟保底](minimal-5min-routine.md)、[20 分钟基础](basic-20min-routine.md)、[30 分钟重建](rebuild-30min-routine.md)的常驻项。
- 原则：**质量优先于次数**，先找到发力感再加量；具体组数次数按自己状态记录、渐进，不照搬别人。

## 自测它弱不弱

- 单腿臀桥左右明显不一样累 / 一侧抬不平；
- 单腿站立时骨盆向一侧掉、躯干歪；
- 跑后总是腰或腿后先累、臀几乎没感觉。

## 训练记录方式

记录动作、组次、是否找到臀的发力感、左右差异、次日感受。把"今天臀终于有感觉了"这种主观突破也记下来。
