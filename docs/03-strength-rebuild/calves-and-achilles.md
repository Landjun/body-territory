# 小腿与跟腱

> 个人训练学习笔记，非医疗建议。动作出现疼痛请停止并参考[红旗信号](../02-running-safety/red-flags.md)。

## 本页目的

讲清楚小腿和跟腱在跑步里如何"既出力又当弹簧"，为什么它们容易紧、容易出问题，以及怎么有针对性地练。这是我当前重点关注的部位之一(小腿易紧)。

## 这个部位在跑步里干什么

小腿后侧主要两块肌肉，都连到跟腱、止于脚跟：

- **腓肠肌**：跨过膝和踝两个关节，膝伸直时更主导。负责快速蹬伸。
- **比目鱼肌**：只跨踝关节，膝屈时更主导。负责持续、耐力性的支撑与缓冲。

它们通过**跟腱**把力传到地面，并且像弹簧一样工作：落地时被拉长储存弹性能，蹬离时回弹释放(牵张-收缩循环 / stretch-shortening cycle)。跑步的"轻盈感"很大程度来自这个弹性回弹。

## 为什么容易紧、容易出问题

- 跑步中小腿-跟腱承受非常多次的反复负荷，每一步都在拉长-回弹；
- 当跑量增长过快、力量或弹性储备不足时，它们最先抗议：**小腿发紧、跟腱晨起僵硬、足底牵连**；
- 比目鱼肌常被忽视(大家只练直腿提踵)，而它在跑步耐力性缓冲里占很大比重；
- 小腿问题常和踝、足底连成一片，所以要和[踝关节稳定](ankle-stability.md)一起看。

## 代表动作

| 动作 | 侧重 | 说明 |
|---|---|---|
| 直腿提踵 | 腓肠肌 | 膝伸直 |
| 屈膝提踵 | 比目鱼肌 | 膝微屈，常被忽略 |
| 离心慢落提踵 | 跟腱韧性 | 重点控制"落"的过程 |
| 单腿提踵 | 力量进阶 | 暴露左右差异 |
| 小腿拉伸(直腿/屈膝) | 柔韧/放松 | 直腿偏腓肠肌、屈膝偏比目鱼肌 |

## 动作详解

### 提踵(双腿 → 单腿)
- **起始**：站立，可扶墙保持平衡；进阶可站台阶边缘增加行程。
- **过程**：缓慢抬到最高(像踮脚尖够东西)，顶端停一下，再**用 2-3 拍缓慢落下**控制离心。
- **要点**：上抬别用弹跳惯性；离心(落)的质量比上抬更重要。
- **变式**：直腿练腓肠肌、屈膝练比目鱼肌，两者都要练。

### 小腿拉伸
- **直腿(偏腓肠肌)**：弓步，后腿伸直、脚跟踩地，身体前移感受后小腿上段。
- **屈膝(偏比目鱼肌)**：同姿势但后膝微屈，感受更靠下、靠近跟腱处。
- **原则**：拉到"紧但不痛"，缓慢、不弹震。

## 常见错误

- **快速弹跳式提踵**：靠惯性弹起落下，跟腱没被有效负荷，也更容易刺激不适。
- **只练向上、不控制下落**：跟腱的韧性主要从可控的**离心**里来。
- **只练直腿、漏掉比目鱼肌**：屈膝提踵被忽略，耐力性缓冲就没练到。
- **已经很紧/不适还硬加量**：紧张是信号，先放松+减量，别在不适上叠负荷。

## 怎么练进训练里

- 提踵适合常驻于各时长模板；
- 紧张明显的日子：以**放松(拉伸、泡沫轴)+ 轻量离心**为主，而不是加量；
- 渐进原则：先双腿、再单腿、再加行程；具体次数按状态记录，渐进，不照搬。

## 自测它弱不弱 / 紧不紧

- 单腿提踵左右次数差异明显，或一侧抬不到顶；
- 晨起或久坐后跟腱/小腿明显僵；
- 跑前就已经紧，跑后更紧、次日加重(→ 参考[小腿紧张复盘](../04-injury-review/calf-tightness-review.md))。

## 训练记录方式

记录提踵组次(直腿/屈膝)、小腿紧张程度(跑前/跑后/次日)、是否伴随足底或跟腱不适。
