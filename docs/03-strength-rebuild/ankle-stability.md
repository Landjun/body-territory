# 踝关节稳定

> 个人训练学习笔记，非医疗建议。动作出现疼痛请停止并参考[红旗信号](../02-running-safety/red-flags.md)。

## 本页目的

讲清楚踝关节作为"落地第一关"的两件事——**活动度**和**稳定性**，为什么两者都重要，以及怎么练、怎么自测。我当前有右脚踝不适，这一页对我尤其相关。

## 这个关节在跑步里干什么

踝其实是两个关节协同：

- **距小腿关节(踝穴)**：负责**背屈**(脚尖往上)和**跖屈**(踮脚尖)。跑步落地缓冲需要足够背屈。
- **距下关节**：负责**内翻/外翻**，在不平地面上微调，吸收侧向的晃动。

此外踝周布满本体感觉受体，是身体判断"我现在脚踩在什么角度的地面上"的关键传感器。

## 为什么对无伤跑步重要

踝同时需要两种看似矛盾的能力：

1. **足够的活动度**(尤其背屈)：背屈不足，落地缓冲就要由膝、髋去代偿；
2. **足够的稳定/控制**：落地瞬间踝若晃动、控制不住内外翻，轻则力量泄漏，重则扭伤。

踝稳定差还会向上影响：落地不稳 → 小腿额外发力代偿(更易紧)、膝承受多余的侧向力。所以踝、踝小腿、膝是一条链。

## 代表动作

| 动作 | 练什么 |
|---|---|
| 单腿平衡(睁眼→闭眼→软垫) | 本体感觉、稳定 |
| 提踵 | 跖屈力量 + 踝控制 |
| 踝绕环 / 脚写字母 | 活动度 |
| 背屈活动度(膝触墙) | 背屈幅度 |
| 弹力带四向(背屈/跖屈/内翻/外翻) | 各方向力量 |

## 动作详解

### 单腿平衡(核心练习)
- **起始**：单脚站稳，膝微屈，目视前方。
- **过程**：保持稳定，脚趾自然，不靠死抓地。先睁眼平地，稳了再加难度。
- **进阶阶梯**：睁眼平地 → 闭眼平地 → 睁眼软垫 → 闭眼软垫 → 加上半身/接抛干扰。
- **要点**：晃动时用**整个踝和小腿微调**找回平衡，这正是在练它。

### 背屈活动度(膝触墙自测兼训练)
- **过程**：脚尖离墙一段距离，膝盖在脚尖正上方向前推去触墙，**脚跟不离地**。能触到就把脚往后挪一点增加难度。
- **意义**：左右脚能触墙的距离差异，反映背屈活动度差异。

### 弹力带四向
- 用弹力带分别对抗背屈、跖屈、内翻、外翻，慢起慢回，补齐各方向薄弱。

## 常见错误

- **用脚趾死抓地代替稳定**：那是代偿，不是踝在控制；放松脚趾重新找。
- **一上来就闭眼/软垫**：超出当前能力反而练成晃动，按阶梯来。
- **只练跖屈(提踵)漏掉背屈活动度**：背屈不足是很多落地代偿的根源。
- **不适时硬练高难度平衡**：右踝不适时降级到睁眼平地或先做活动度。

## 怎么练进训练里

- 单腿平衡几乎零成本，适合刷牙、等水开时顺手做；
- 不适期：以**活动度 + 低难度平衡**为主，不上高难度；
- 渐进按阶梯走，记录能稳住的时长和难度。

## 自测

- 膝触墙左右距离差明显(背屈不对称)；
- 单腿闭眼站立左右时长差异大；
- 落地或下楼时踝有"要崴"的不安定感。

## 训练记录方式

记录单腿平衡时长/难度、膝触墙距离左右差、右踝不适评分与变化(关联[右脚踝复盘](../04-injury-review/ankle-pain-review.md))。
