## 怎么把动作组成一堂课

一个简单、好用的排课骨架：

1. **热身/活动度**(踝绕环、髋部活动、原地踏步)；
2. **激活**(臀桥、蚌式)——先把该工作的肌肉叫醒；
3. **力量**(提踵、静蹲、台阶下控)——主菜；
4. **稳定/平衡**(平板、死虫、单腿平衡)——练抗动与控制；
5. **放松/拉伸**(小腿拉伸、髋部活动)——收尾。

时间不够时，按这个顺序"砍尾巴"：先保激活+力量，再加稳定，最后才是拉伸。成套模板见 [5 分钟](minimal-5min-routine.md) / [20 分钟](basic-20min-routine.md) / [30 分钟](rebuild-30min-routine.md)。

## 安全与渐进原则

- **质量优先于次数/负荷**：先找发力感与对线，再谈加量；
- **渐进**：双腿→单腿、小幅→大幅、稳定面→不稳定面，一次只加一个难度；
- **疼痛优先**：任何动作引发疼痛或次日加重，降级或停止并复盘；
- 具体组数次数请按自己状态记录、逐步调整，**不照搬别人的数字**。
