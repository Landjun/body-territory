# 动作库总表

把当前阶段常用的基础动作集中在一处，方便排课和查阅。每个动作的详细要点见对应部位文件。

## 怎么用这张表

- 排课时从表里挑动作，组合成 5 / 20 / 30 分钟模板；
- 每个动作都遵循"质量优先于次数"。

## 费曼讲解问题

- 这套动作库覆盖了跑步需要的哪些能力？
- 为什么基础动作就足以支撑当前阶段？
- 它如何帮助我减少受伤？
- 它如何服务网络安全主线？
- 今天我能做的最小动作是什么？
