## 为什么当前阶段力量比跑量更重要

- 当前阶段力量训练比盲目跑量更重要；
- 这不是背叛跑步；
- 这是为了未来更安全地跑；
- 跑步不是只有心肺；
- 跑步还需要臀、腿、核心、踝、小腿、膝稳定；
- 力量训练是安全跑者的基础设施。
