# 膝关节控制

> 个人训练学习笔记，非医疗建议。动作出现疼痛请停止并参考[红旗信号](../02-running-safety/red-flags.md)。

## 本页目的

讲清楚一个反直觉的事实：**膝痛常常不是膝自己的问题**，膝的稳定主要由上游的臀和下游的踝决定。理解这一点，才知道"练膝"该练什么。

## 这个关节在跑步里干什么

膝主要是一个**屈伸 + 缓冲**的铰链关节，本身的左右旋转/侧移空间很小。跑步中它的角色是：

- 落地时**离心缓冲**(股四头肌控制屈膝)，吸收冲击；
- 蹬伸时配合伸髋伸膝把人送出去；
- 全程保持**对线**：理想情况下膝大致朝向脚尖方向运动。

髌骨在股骨的"轨道"里滑动，**股四头肌(尤其股内侧)和髋的稳定**共同决定它走得顺不顺。

## 为什么"膝问题"要去查臀和踝

膝夹在髋和踝中间，上下游一旦失稳，膝就被迫承受它不擅长的侧向力：

- **臀中肌弱 / 髋不稳** → 支撑时髋内收、**膝内扣(knee valgus)** → 髌骨轨迹和髂胫束受力异常；
- **踝背屈不足 / 不稳** → 落地缓冲不够、力往上传到膝；
- 于是"膝痛"很多时候是**上下游的下游表现**。练膝的重点常常落在**练臀 + 练踝 + 改善落地对线**。

## 代表动作

| 动作 | 作用 |
|---|---|
| 靠墙静蹲 | 股四头耐力、膝对线 |
| 自重深蹲(可控幅度) | 蹲模式、膝-脚尖对线 |
| 台阶下控(step-down) | 离心缓冲、单腿对线 |
| 单腿平衡 | 下肢整体控制 |
| 臀桥 / 蚌式 | 上游臀部(间接护膝) |

## 动作详解

### 靠墙静蹲
- **起始**：背贴墙，脚离墙一段，下滑到膝屈约 90°(或无痛范围内)。
- **要点**：膝**朝脚尖方向、不内扣**、不超过脚尖太多；保持核心收紧、呼吸顺畅。
- **退阶**：减小屈膝角度、缩短时间。

### 台阶下控(step-down)
- **过程**：单腿站台阶上，缓慢屈膝让另一脚轻点地面再起来，**全程盯住支撑膝不往内倒**。
- **意义**：直接训练落地最需要的"单腿离心 + 对线控制"。

### 自重深蹲
- 蹲下时髋膝同步、膝随脚尖方向；起身用臀腿，别塌腰。

## 常见错误

- **膝内扣**：下蹲/落地时膝往内倒——最该盯的对线问题，常因臀弱。
- **只盯着膝练**：忽略臀和踝，治标不治本。
- **速度过快、缺乏离心控制**：缓冲能力练不到。
- **疼痛状态做大负荷蹲跳**：膝不适期应降到无痛范围的静蹲/对线练习。

## 怎么练进训练里

- 膝相关训练核心是"**对线 + 离心控制**"，而非堆重量；
- 把臀(上游)和踝(下游)的练习视为护膝的一部分；
- 不适期：无痛范围静蹲 + 单腿平衡，避免冲击性动作。

## 自测

- 对镜单腿下蹲，支撑膝是否往内倒、是否能与脚尖同向；
- 下楼梯是否某侧膝不稳或不适；
- 静蹲时是膝周还是大腿前在工作(应是大腿前/臀，而非膝内压痛)。

## 训练记录方式

记录静蹲时长、单腿下蹲对线是否良好、膝不适评分与诱发动作。
