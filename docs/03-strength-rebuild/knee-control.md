# 膝关节控制

## 作用

膝关节本身主要是"传导"和"缓冲"，它的稳定很大程度上取决于上游的臀和下游的踝、小腿。

## 为什么对跑步重要

膝痛往往不是膝自己的问题，而是臀无力、踝不稳、落地方式不佳的下游表现。改善膝控制，重点常在臀和踝。

## 代表动作

- 靠墙静蹲（控制膝在脚尖方向、不内扣）；
- 臀桥与蚌式（强化上游臀部）；
- 单腿平衡（改善下肢整体控制）。

## 如何做（要点）

- 靠墙静蹲：背贴墙，膝不超过脚尖、不内扣，保持稳定时间；
- 关注膝是否与脚尖同向。

## 常见错误

- 下蹲时膝盖内扣；
- 只盯着膝盖练，忽略臀和踝；
- 疼痛状态下做深蹲类高负荷动作。

## 训练记录方式

记录静蹲时长、膝是否内扣、膝不适评分。
