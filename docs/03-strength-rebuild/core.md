# 核心

## 作用

核心（躯干深层与腹背肌群）在跑步中负责保持躯干稳定，让臀腿的力量有效传导，不被晃动浪费。

## 为什么对跑步重要

核心不稳，跑姿容易塌、骨盆容易晃，能量在传导中流失，也更容易引发腰背和下肢代偿。

## 代表动作

- 平板支撑；
- 死虫；
- 臀桥（兼顾核心与臀）。

## 如何做（要点）

- 平板支撑：身体成直线，收紧腹部和臀，不塌腰不翘臀；
- 死虫：仰卧，对侧手脚缓慢伸展，全程腰背贴地、核心收紧。

## 常见错误

- 平板塌腰或撅臀；
- 死虫时腰离地、靠惯性甩动；
- 只追求时长，失去躯干稳定。

## 训练记录方式

记录平板时长、死虫次数、腰背是否能保持稳定。

## 费曼讲解问题

- 核心对跑步的作用用一句话怎么讲？
- 为什么核心不稳会"浪费"臀腿力量？
- 它如何帮助我减少受伤？
- 它如何服务网络安全主线？
- 今天我能做的最小动作是什么？
