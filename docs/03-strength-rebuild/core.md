# 核心

> 个人训练学习笔记，非医疗建议。动作出现疼痛请停止并参考[红旗信号](../02-running-safety/red-flags.md)。

## 本页目的

纠正一个常见误解：跑步的核心训练，主要不是"练出腹肌、做很多卷腹"，而是练**抗动**——在跑动中保持躯干稳定、不让力量从晃动里泄漏掉。

## 核心到底指什么

核心是包裹躯干的一组肌肉，像一个"圆桶"：

- 前面：腹直肌、腹横肌(深层、像束腰带一样收紧腹腔)；
- 侧面：腹内/外斜肌；
- 后面：竖脊肌、多裂肌；
- 上下盖：膈肌(上)和盆底肌(下)。

它们协同制造**腹内压和躯干刚性**，让脊柱在受力时保持稳定。

## 跑步里核心的真正任务:抗动，而不是制造动作

跑步时躯干理想状态是**相对稳定**的，核心的工作是抵抗各种让你晃的力：

- **抗伸展**：防止跑累时塌腰、骨盆前倾；
- **抗旋转**：每一步手脚交替会带来旋转力，核心要稳住躯干不乱扭；
- **抗侧屈**：单腿支撑时防止躯干往一侧倒。

核心不稳的代价：骨盆晃、躯干塌、臀腿辛苦产生的力在传导中泄漏，腰背被迫代偿 → 又累又容易腰背不适。

## 代表动作(以"抗动"为主)

| 动作 | 抗什么 |
|---|---|
| 平板支撑 | 抗伸展 |
| 侧平板 | 抗侧屈 |
| 死虫 | 抗伸展(四肢移动时躯干不动) |
| 鸟狗 | 抗旋转/抗伸展 |
| 臀桥 | 兼顾核心 + 臀 |

## 动作详解

### 平板支撑
- **要点**：头-背-臀-腿成一条直线，**收腹收臀**、骨盆微后倾，别塌腰也别撅臀。
- **质量 > 时间**：能保持直线的 20 秒，强过塌着腰撑很久。

### 死虫
- **起始**：仰卧，手臂指向天花板，髋膝屈 90°。
- **过程**：对侧手脚缓慢伸展放低，**全程腰背贴地、核心收紧**，再换边。
- **关键**：腰一旦离地就说明超出当前能力，缩小幅度。

### 鸟狗
- **过程**：四点支撑，伸出对侧手和腿到与躯干齐平，**保持骨盆不歪不转**，稳住再收回。

## 常见错误

- **塌腰 / 撅臀**：平板失去直线，练的就不是核心稳定。
- **憋气**：核心训练要能正常呼吸，憋气是代偿。
- **靠惯性甩动**(死虫/鸟狗)：动作要慢而控，重点是躯干"不动"。
- **只追时长/数量**：失去躯干刚性后继续撑没有意义。

## 怎么练进训练里

- 核心适合高频、短时；
- 重点选"抗动"类动作而非大量卷腹；
- 渐进方式：先稳住基础姿势，再延长时间/增加幅度/增加不稳定。

## 自测

- 平板时是否很快塌腰；
- 死虫伸腿时腰是否离地；
- 跑后是否经常腰背先累(可能核心抗伸展不足)。

## 训练记录方式

记录平板能保持直线的时间、死虫/鸟狗能否保持腰背稳定、跑后腰背疲劳变化。
