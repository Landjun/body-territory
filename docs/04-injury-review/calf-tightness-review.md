# 小腿紧张复盘

> 这是个人训练复盘模板，不是医学诊断。

> 小腿紧不是小问题，它可能是身体提醒我：负荷、恢复、力量或动作模式需要调整。

## 本页目的

把"小腿紧"这个常见信号结构化，找出它和负荷、鞋、跑步机、其他部位的关系。

## 记录项

- 小腿紧张出现时间：
- 是否跑前已有：
- 是否跑后加重：
- 是否次日加重：
- 是否与跑量增加有关：
- 是否与跑步机有关：
- 是否与鞋有关：
- 是否伴随足底、跟腱、踝不适：
- 是否适合跑步：
- 是否改成力量和拉伸：

## 决策提示

- 小腿明显紧或不适时，优先改为快走 + 拉伸 + 轻力量；
- 紧张反复出现时，回看跑量是否增长过快、提踵力量是否不足。
