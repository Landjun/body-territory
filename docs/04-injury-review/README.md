> 安全跑者不逃避疼痛，也不崇拜疼痛。
> 安全跑者把疼痛当作系统反馈。

## 本模块在做什么

把每一次疼痛、紧张、不适，变成一次结构化复盘，找出诱因、决定降级策略、明确恢复条件，避免同样的问题反复出现。

## 复盘维度

1. 疼痛位置；
2. 疼痛等级；
3. 出现时间；
4. 是否训练中出现；
5. 是否次日加重；
6. 最近 7 天训练量；
7. 睡眠；
8. 是否突然加量；
9. 是否做速度；
10. 是否忽略热身；
11. 是否压力大；
12. 今日降级策略；
13. 何时恢复跑步。
