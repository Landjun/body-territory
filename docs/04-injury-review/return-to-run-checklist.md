# 回归跑步清单

> ⚠️ 个人训练复盘方法,非医疗诊断;疼痛持续/加重或不确定时,请咨询医生或康复师后再回归。

## 本页目的

在伤病或不适后，用一份清单判断"能不能重新开始跑"，避免太早回归再次受伤。

## 恢复跑步前至少满足

- 疼痛 0-2；
- 不影响步态；
- 次日不加重；
- 可以完成基础力量动作；
- 快走无加重。

## 回归后的前几次

- 先从低强度开始；
- 不做速度；
- 不加量；
- 记录次日反馈。

## 适用场景

- 一次降级/休息后准备恢复跑步时；
- 一段时间没跑、重新开始时。

## 核心原则

回归是渐进过程，不是一次"满血复活"。任何一条不满足，就再等一等。

## 最小行动

对照上面的条件，逐条给自己打勾，缺一条就先不跑。

## 费曼讲解问题

- 回归跑步的最低门槛是什么？
- 为什么回归后还要"不加量、不做速度"？
- 它如何帮助我减少受伤？
- 它如何服务网络安全主线？
- 今天我能做的最小动作是什么？
