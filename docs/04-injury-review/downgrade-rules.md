# 降级规则

> 降级不是撤退，而是保护长期系统。

## 本页目的

明确"什么情况降几级、怎么降"，让降级成为有规则的动作，而不是临场情绪。

## 降级路径

- A → B；
- B → C；
- C → D；
- D → E；
- E → 休息。

## 触发因素

- 睡眠不足；
- 能量低；
- 疼痛；
- 次日加重；
- 压力大；
- 时间不足；
- 情绪想硬撑。

> 多个因素可叠加；多个因素同时出现时，可一次降多级。

## 适用场景

每次跑前自检后，如果触发了任一因素，按路径降级。

## 核心原则

- 降级保护的是长期连续性；
- 今天降一级，是为了明天还能继续训练。

## 最小行动

如果今天触发了降级因素，按路径选定新的等级并执行。

## 费曼讲解问题

- 降级规则用一句话怎么讲？
- 为什么说"降级不是撤退"？
- 它如何帮助我减少受伤？
- 它如何服务网络安全主线？
- 今天我能做的最小动作是什么？
