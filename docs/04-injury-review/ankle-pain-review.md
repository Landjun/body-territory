# 右脚踝不适复盘

> 这是个人训练复盘模板，不是医学诊断。

## 本页目的

针对当前的右脚踝不适，建立一份可重复使用的复盘清单，追踪诱因与恢复。

## 记录项

- 疼痛位置：
- 疼痛等级 0-10：
- 出现时间：
- 跑步中是否加重：
- 次日是否加重：
- 是否影响步态：
- 最近 7 天训练：
- 鞋子 / 跑步机 / 地面：
- 小腿是否紧张：
- 是否做过力量：
- 是否做过拉伸：
- 今天是否需要降级：

## 决策原则

- 疼痛 3-4：不跑；
- 疼痛 5+：停止跑跳；
- 步态改变：停止跑步；
- 次日加重：降级；
- 优先做快走、轻力量、拉伸和复盘。
