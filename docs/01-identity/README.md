# 身份定位

在讲任何训练方法之前，先讲身份。因为方法会变，身份不变。

## 为什么这个仓库要先讲身份

训练方法、配速、动作都会随阶段变化，但"我为什么训练、训练服务什么"必须先定下来。
身份定位决定了：状态不好时我会降级而不是硬撑；身体和主线冲突时我知道谁服务谁。

## 为什么跑步不能反客为主

我曾经很喜欢跑步，也曾因为受伤两年没踏入操场。
跑步可以是热爱，但当前阶段它的定位是**身体底盘**，不是人生一级主线。
网络安全才是圆心。跑步、力量、睡眠、康复都服务于让我有更稳定的身体走更长的安全之路。

## 为什么身体底盘服务网络安全主线

长期学习需要稳定的身体；稳定的身体需要训练、睡眠、恢复和节奏。
所以身体训练不是抢主线时间，而是为主线提供基础设施。
判断标准很简单：一个身体行为如果让我更可持续地学习和工作，它就服务主线；如果它在消耗我、让我受伤、让我逃避，它就偏离了主线。

## 为什么"安全跑者"不是跑步博主标签

"安全跑者"不是一个内容人设，而是一个长期系统身份：

> 一个用身体的纪律映照安全的纪律的人。

它强调的是边界、反馈、复盘和长期一致性——这些恰恰也是网络安全的底层素养。
