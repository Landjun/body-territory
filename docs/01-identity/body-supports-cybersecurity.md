# 网络安全是目的，身体是基础设施

## 本页目的

把身体训练放回人生系统的正确位置：支撑主线，而不是抢占主线。

## 核心逻辑

- 网络安全主线需要长期学习；
- 长期学习需要稳定身体；
- 稳定身体需要训练、睡眠、恢复、节奏；
- 跑步和力量不是抢主线时间；
- 但跑步不能成为新的逃避；
- 当前阶段跑步正确定位是：身体底盘维护。

## 是否服务主线判断表

| 行为 | 是否服务主线 | 判断 |
|---|---|---|
| 安全跑 20 分钟 | 是 | 提升身体稳定性 |
| 疼痛硬跑 10 公里 | 否 | 消耗主线身体资源 |
| 10 分钟力量训练 | 是 | 建立长期底盘 |
| 训练后写复盘 | 是 | 形成方法论资产 |
| 过度刷装备 | 不一定 | 若不产出训练效果，可能是逃避 |

## 适用场景

- 纠结"今天该学习还是该跑步"时，用这张表判断；
- 想用训练逃避主线任务时，先识别它是支撑还是逃避。

## 核心原则

- 身体行为以"是否让主线更可持续"为最终标尺；
- 支撑型身体投入要保留，逃避型身体投入要警惕。

## 最小行动
