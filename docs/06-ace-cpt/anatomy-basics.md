# 解剖学基础

> 个人学习笔记，非官方教材、非医疗建议。

## 大白话解释

解剖学就是"身体由哪些零件组成、它们怎么连在一起、怎么配合产生动作"。对跑步来说，重点是搞懂骨骼、关节、肌肉这三者如何协作完成"落地—支撑—蹬伸"这一连串动作。

## 核心知识点

### 1. 三个运动平面
身体的动作可以用三个互相垂直的平面来描述：

- **矢状面**：把身体分成左右，前后方向的动作(屈/伸)。跑步主要发生在这个面——抬腿、蹬腿都是。
- **额状面**：把身体分成前后，左右方向的动作(外展/内收)。跑步里这个面更多是"被动抗住"——比如臀中肌防止骨盆侧塌。
- **水平面**：旋转类动作。跑步里躯干的抗旋转就发生在这个面。

> 一个反直觉但重要的点：跑步看似只在矢状面"前进"，但**额状面和水平面的稳定**(防侧塌、防乱转)恰恰是无伤的关键。

### 2. 关节与活动度
关节是骨与骨相接、产生活动的地方。不同关节天生"职责不同"：有的偏向**活动度**(如踝、髋、胸椎)，有的偏向**稳定**(如膝、腰椎)。当该灵活的不灵活(比如踝背屈受限)，本该稳定的关节(膝、腰)就被迫去代偿——这是很多劳损的来源。

### 3. 肌肉的四种角色
同一个动作里，肌肉分工不同：

- **主动肌(agonist)**：主要发力的；
- **拮抗肌(antagonist)**：相反方向、负责控制和平衡的；
- **协同肌(synergist)**：帮忙的；
- **稳定肌(stabilizer)**：固定住某部分，让动作有支点的。

理解这点就明白：练好"稳定肌"(如核心、臀中肌)，主动肌才能高效出力。

### 4. 三种收缩方式
- **向心**：肌肉缩短发力(提踵抬起、蹬地)；
- **离心**：肌肉拉长的同时控制(落地缓冲、提踵慢落)——**跑步减伤和力量增长的关键常在离心**；
- **等长**：长度不变地用力(靠墙静蹲、平板支撑)。

### 5. 跑步相关的主要肌群
臀(推进+稳定)、股四头(缓冲、伸膝)、腘绳肌(屈膝、伸髋)、小腿+跟腱(蹬伸、弹性)、核心(躯干稳定)。详见[力量重建](../03-strength-rebuild/README.md)各部位页。

### 6. 动力链(kinetic chain)
身体是一条链：地面反作用力从脚→踝→膝→髋→躯干层层传导。任一环薄弱或失稳，相邻环就要代偿。这解释了为什么"膝痛去练臀和踝"。

## 与跑步基座的关系

解剖学是力量重建里"为什么练这个部位"的底层答案：知道臀中肌在额状面稳定骨盆、知道离心收缩负责缓冲，才能有的放矢地排课，而不是盲目跟练。

## 与安全跑者的关系

懂结构 → 懂代偿 → 懂边界。安全跑者用解剖知识判断"这个不适可能来自哪条链的哪一环"，从而做出更聪明的降级和复盘。

## 训练应用

把[动作库](../03-strength-rebuild/movement-library.md)里每个动作，标注它练的是哪个肌群、哪种收缩、哪个平面的稳定。

## 费曼讲解问题

- 三个运动平面分别是什么？跑步主要在哪个面，但哪些面负责"稳"？
- 主动肌/拮抗肌/协同肌/稳定肌怎么分工？
- 向心、离心、等长收缩各举一个跑步/力量例子。
- 什么是动力链？它怎么解释"膝痛练臀踝"？
- 这块知识今天能改变我哪一个训练决定？
