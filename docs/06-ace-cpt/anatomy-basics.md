# 解剖学基础

## 大白话解释

解剖学就是"身体由哪些零件组成、它们怎么连在一起发力"。对跑步来说，重点是骨骼、关节、肌肉怎么配合完成一个动作。

## 核心知识点（学习骨架）

- 骨骼与关节：身体的支架和活动枢纽；
- 肌肉：通过收缩产生动作，只能"拉"不能"推"；
- 主要跑步相关肌群：臀、股四头、腘绳、小腿、核心；
- 解剖方位与动作术语：屈/伸、内收/外展等。

## 与跑步基座的关系

知道哪块肌肉负责什么，才能理解力量重建里为什么练臀、练踝、练小腿。

## 与安全跑者的关系

理解结构，才能理解代偿——为什么膝痛常去练臀，为什么踝不稳会牵连小腿。

## 训练应用

把[动作库](../03-strength-rebuild/movement-library.md)里的每个动作，对应到它主要练的肌群。
