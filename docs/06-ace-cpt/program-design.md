# 训练计划设计

> 个人学习笔记，非官方教材、非医疗建议。

## 大白话解释

训练计划设计就是"把一堆动作、强度和休息，按目标、时间和恢复，组织成一份能执行、能进步、又不受伤的安排"。它回答的是"先练什么、练多少、怎么一周周变强"。

## 核心知识点

### 1. 先做需求分析
开练之前先问清楚：目标是什么(恢复/健康/某个比赛)？现状如何(伤病史、薄弱环节、可用时间)？有什么限制(右脚踝、小腿易紧)？计划是"为目标和现状量身定的"，不是抄来的。

### 2. FITT:计划的四个基本旋钮
- **F 频率**：一周练几次；
- **I 强度**：多累(用心率、谈话测试、自觉用力 RPE 来把握，而不是死磕配速)；
- **T 时间**：每次多久；
- **T 类型**：跑、快走、力量、拉伸……不同类型练不同能力。

### 3. 训练量与强度的平衡
量(总做了多少)和强度(多累)不能同时往上猛冲。无伤的做法是：大部分训练是低强度打底，高强度只占小比例，且不在疲劳/疼痛时安排。

### 4. 动作顺序
一般原则：复杂的、需要专注和爆发的放前面，简单的、耐力性的放后面。对应到我的[排课骨架](../03-strength-rebuild/movement-library.md)：热身→激活→力量→稳定→放松。

### 5. 渐进与周期化
- **渐进超负荷**：逐周可控地加一点点；
- **周期化**：把训练分成阶段(打基础→提升→保持→恢复)，强度和量按阶段起伏，给身体留出适应和恢复的节奏。详见[周期化原理](../../02-injury-free-running/periodization/training-periodization.md)。

### 6. 把恢复写进计划
好的计划自带"刹车"——休息日、减量周、降级规则都是计划的一部分，而不是计划失败才用的补救。

### 7. 个体化与再评估
计划不是一次定终身。按反馈(疼痛、次日状态、睡眠)定期再评估、再调整。

## 与跑步基座的关系

我的 A-E 训练等级 + 5/20/30 分钟模板，就是一份简化的、以"安全恢复"为目标的计划设计：用 FITT 思路，根据每天状态选强度和类型，把恢复(降级、保底)直接编进系统。

## 与安全跑者的关系

会设计计划的人，不靠意志力硬扛，而靠结构托住自己。安全跑者的计划里，"刹车"和"油门"一样重要。

## 训练应用

用 FITT 检查我当前一周：频率、强度、时间、类型是否匹配"安全恢复"这个目标？哪一项过头了？
