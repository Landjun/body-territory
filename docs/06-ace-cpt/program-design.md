# 训练计划设计

## 大白话解释

训练计划设计就是"把一堆动作和强度，按目标、时间和恢复，组织成一份可执行的安排"。

## 核心知识点（学习骨架）

- 目标导向：先有目标，再有计划；
- FITT：频率、强度、时间、类型；
- 渐进超负荷：逐步加压；
- 周期化：分阶段安排强度与恢复；
- 个体化：按当前水平和限制调整。

## 与跑步基座的关系

我的 A-E 训练等级、5/20/30 分钟模板，本质上就是一份简化的、以安全恢复为目标的计划设计。

## 与安全跑者的关系

好的计划自带"刹车"——它把恢复和降级也写进去，而不是只堆强度。

## 训练应用

用 FITT 检查我当前一周的安排：频率、强度、时间、类型是否匹配"安全恢复"这个目标。

## 费曼讲解问题

- 训练计划设计用一句话怎么讲？

- 它如何帮助我减少受伤？
- 它如何服务网络安全主线？
- 今天我能做的最小动作是什么？
