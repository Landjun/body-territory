# 训练适应

## 大白话解释

训练适应就是"身体被合适的压力刺激后，休息时变得更强"。训练当下其实是变弱，恢复之后才变强。

## 核心知识点（学习骨架）

- 超负荷：给身体一个略高于习惯的刺激；
- 恢复与超量恢复：变强发生在休息期；
- 渐进：刺激要逐步增加，太快会受伤；
- 可逆性：不练会退回；
- 个体差异：每个人适应速度不同。

## 与跑步基座的关系

这是负荷管理、10% 原则、降级规则背后的核心原理——"压力 + 恢复 = 进步，压力 - 恢复 = 受伤"。

## 与安全跑者的关系

理解适应，就明白降级和休息不是浪费，而是变强的必要环节。

## 训练应用

每次加量前，先问"上一次的刺激恢复了吗"（看次日反馈）。

## 费曼讲解问题

- 训练适应用一句话怎么讲？
- 为什么"变强发生在休息时"？
- 它如何帮助我减少受伤？
- 它如何服务网络安全主线？
- 今天我能做的最小动作是什么？
