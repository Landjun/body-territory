# 运动科学学习地图

> 个人学习笔记，非官方教材、非医疗建议。

## 大白话解释

运动科学就是"身体在运动时发生了什么、长期会怎么变"的学问。学懂它，我就能**解释自己每一次训练的目的**，而不是照搬别人的计划、跟着感觉走。这张地图是我整个身体底盘系统的"科学后台"。

## 这张地图怎么读

把运动科学拆成几块，每块回答一个问题，彼此衔接：

| 板块 | 它回答的问题 | 对应笔记 |
|---|---|---|
| 解剖学 | 身体由什么组成、怎么发力 | [anatomy-basics](anatomy-basics.md) |
| 运动生理学 | 运动时心肺、能量、肌肉怎么反应 | 待补充 |
| 训练适应 | 长期训练让身体怎么变强 | [training-adaptation](training-adaptation.md) |
| 体能要素 | 有氧、力量、柔韧、平衡各是什么 | 待补充 |
| 动作模式 | 人的基本动作有哪些 | [动作与体态(旧)](../../03-ace-cpt/movement-and-posture-training.md) |
| 风险筛查 | 开练前要排除什么 | [risk-awareness](risk-awareness.md) |
| 计划设计 | 怎么组织成一份训练 | [program-design](program-design.md) |
| 沟通与行为改变 | 怎么把科学变成能坚持的行动 | [行为改变(旧)](../../03-ace-cpt/behavior-change-communication.md) |

## 一条主线把它们串起来

```text
懂结构(解剖) → 懂反应(生理) → 懂变强机制(适应)
        → 选对刺激(体能要素 + 动作模式)
        → 先排风险(筛查) → 组织成计划(设计)
        → 让人坚持(沟通) → 在我身上落地(训练应用)
```

## 学习方法:费曼优先

我是"边看边讲"型，所以这张地图的每一块都按这个流程啃：

1. 对着标题先讲一遍 → 2. 暴露卡点 → 3. 回资料补学 → 4. 再讲 → 5. 写成[费曼卡片](feynman-cards.md)。

## 与跑步基座的关系

跑步安全、力量重建、伤病复盘里的每一个判断，背后都能在这张地图里找到原理。学地图不是为了考证，而是为了让我的身体系统"知其所以然"。

## 与安全跑者的关系

懂原理才能在边界内安全加压，而不是凭感觉冒进。地图越清晰，决策越稳。

## 训练应用

每学完一块，回到对应的训练文件，问一句："这条原理改变了我哪一个具体的训练决定？"

## 费曼讲解问题

- 运动科学这张地图分成哪几块？各回答什么问题？
- 这条"懂结构→懂反应→懂变强→选刺激→排风险→设计→坚持→落地"的主线，能不能一口气讲下来？
- 为什么对我来说"边看边讲"比"先整理再输出"更高效？
- 学这张地图和"考证"有什么本质区别？
- 这块知识今天能改变我哪一个训练决定？
