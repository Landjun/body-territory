# 运动科学学习地图

## 大白话解释

运动科学就是"身体在运动时发生了什么、长期会怎么变"的学问。把它学懂，就能解释自己每一次训练的目的，而不是照搬别人的计划。

## 核心知识点（学习骨架）

- 解剖学：身体由哪些结构组成、怎么发力；
- 运动生理学：运动时心肺、能量、肌肉怎么反应；
- 训练适应：长期训练让身体发生哪些改变；
- 心肺、肌力、灵活性：体能的几个维度；
- 动作模式：人体的基本动作；
- 风险筛查：开始训练前要排除什么风险；
- 计划设计：怎么把这些组织成一份训练；
- 沟通：怎么把科学变成对方能执行的行动。

## 与跑步基座的关系

这张地图是我整个身体底盘系统的"科学后台"——跑步安全、力量重建、伤病复盘里的判断，背后都能在这里找到原理。

## 与安全跑者的关系

懂原理，才能在边界内安全加压，而不是凭感觉冒进。

## 训练应用

每学一块，就回到对应的训练文件，问"这条原理改变了我哪一个训练决定"。
