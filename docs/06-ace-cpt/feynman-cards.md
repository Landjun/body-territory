# 费曼知识卡片

把 ACE-CPT 的知识点拆成一张张可以"开口就讲"的小卡片。每张卡片只讲一个概念，先一句话，再展开。

## 卡片模板

```markdown
### 概念名

- 一句话讲清：
- 三点展开：1. / 2. / 3.
- 跑步里的例子：
- 和安全跑者的关系：
- 卡点：
```

每学完一个概念，就照模板写一张卡片，然后对着[手表录音笔](../07-feynman-output/watch-recorder-workflow.md)讲一遍。
