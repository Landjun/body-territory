# 跑步健康如何费曼学习

## 本页目的

给出一个把跑步/运动科学知识点"讲懂"的固定套路。

## 五步套路

1. 拿一个知识点；
2. 先一句话讲；
3. 再三点展开；
4. 再举一个训练例子；
5. 再讲给零基础的人；
6. 最后转化为一个训练动作或决定。

## 核心原则

讲得出 + 举得出例子 + 能落到一个动作，才算真的学会。

## 最小行动

挑一个我已经"以为懂"的概念，按五步讲一遍，记录卡在第几步。

## 费曼讲解问题

- 这套五步法用一句话怎么讲？
- 为什么最后一步要"转化为动作"？
- 它如何帮助我减少受伤？
- 它如何服务网络安全主线？
- 今天我能做的最小动作是什么？
