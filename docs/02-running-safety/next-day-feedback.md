# 次日反馈

## 本页目的

把"训练好不好"的评判标准，从当天感觉挪到次日反馈。

## 核心观点

- 训练当天感觉不代表恢复成功；
- 次日疼痛、僵硬、疲劳、精神状态更重要；
- 次日加重说明前一天负荷可能过大；
- 下一次训练要降级。

## 次日要看的信号

- 目标部位疼痛是否加重；
- 是否比平时更僵硬；
- 整体疲劳和精神状态；
- 是否影响走路步态。

## 适用场景

每次训练后的第二天早晨，花 30 秒做一次次日自检。

## 核心原则

次日加重 = 上一次负荷过大 = 下一次降级并复盘。

## 最小行动

明早起床后，给昨天训练的部位打个次日疼痛分，记录到日志。

## 费曼讲解问题

- 次日反馈用一句话怎么讲？
- 为什么它比"训练中咬牙完成"更重要？
- 它如何帮助我减少受伤？
- 它如何服务网络安全主线？
- 今天我能做的最小动作是什么？
