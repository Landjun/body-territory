# 次日反馈

> 个人训练学习笔记，非医疗建议。

## 本页目的

把"这次训练到底合不合适"的评判标准，从**当天的感觉**挪到**第二天的反馈**——这是判断负荷是否过量、最可靠也最容易被忽略的信号。

## 核心观点

- 训练当天感觉好，不代表恢复成功；
- 次日的疼痛、僵硬、疲劳、精神状态，更能说明真相；
- 次日加重，说明前一天负荷可能过大；
- 这时下一次训练要降级，而不是"再扛一下"。

## 为什么次日比当天更准

训练当天，肾上腺素、内啡肽、"今天状态不错"的兴奋感，会**掩盖**真实的负荷代价。很多损伤是"当时不觉得、第二天才知道"。睡了一觉、激素回落之后，身体给出的反馈才接近真实账单。

## 次日早上花 30 秒看四件事

1. **目标部位疼痛**：比昨天前是否加重(尤其右脚踝、小腿)；
2. **僵硬度**：比平时明显更僵吗；
3. **整体疲劳/精神**：是恢复性的轻松，还是被掏空的萎靡；
4. **步态**：走路是否正常，有没有不自觉的保护性姿势。

## 把次日反馈接回决策

- 次日**没加重、精神好** → 上次负荷合适，可按计划进行或小步渐进；
- 次日**加重 / 异常疲劳 / 步态变化** → 降级 + 复盘诱因(参考[降级规则](../04-injury-review/downgrade-rules.md)、[伤病复盘](../04-injury-review/README.md))。

这条反馈回路，就是[训练适应](../06-ace-cpt/training-adaptation.md)里"压力+恢复=进步"的日常落地。

## 适用场景

每次训练后的第二天早晨；尤其在加量、尝试新动作、或本就有不适信号之后。

## 最小行动

明早起床后，给昨天训练的部位打一个次日疼痛分，连同精神状态记进日志。

## 费曼讲解问题

- 为什么次日反馈比当天感觉更准？(激素的作用)
- 次日要看哪四件事？
- 次日加重时正确的下一步是什么？
- 它和"压力+恢复=进步"有什么关系？
- 今天我能做的最小动作是什么？
