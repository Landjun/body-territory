# 疼痛分级：训练计划必须服从身体信号

> 个人训练判断参考，非医疗诊断。出现红旗信号请参考[红旗信号](red-flags.md)并咨询专业人士。

## 本页目的

把"疼痛"从一个模糊、容易被意志力压下去的感觉，变成一个能驱动训练决策的、分级的信号。

## 分级表

| 疼痛等级 | 含义 | 能否跑 | 建议 |
|---|---|---|---|
| 0 | 无痛 | 可以 | 正常低强度训练 |
| 1-2 | 轻微不适 | 谨慎可以 | 不做速度，不加量 |
| 3-4 | 明显不适 | 不建议 | 快走、力量、拉伸 |
| 5-6 | 中高风险 | 不跑 | 停止跑跳，观察 |
| 7-10 | 高风险 | 不跑 | 必要时咨询专业人士 |

> 当下能忍不代表身体能恢复。
> 次日是否加重，比训练中是否咬牙完成更重要。

## 怎么给疼痛打分(让评分更一致)

打分时尽量固定几个维度，前后才可比：

- **位置**：具体在哪(右脚踝外侧 / 小腿中段 / 膝前)；
- **性质**：酸、紧、钝痛、还是尖锐刺痛(**尖锐刺痛要格外警惕**)；
- **出现时机**：跑前就有 / 跑中出现 / 跑后 / 次日；
- **随活动的变化**：热身后减轻(相对安全)还是越跑越重(危险)；
- **影响**：是否改变了步态(改变步态是高风险信号)。

## 为什么"忍痛跑完"是错的

疼痛是身体的报警，不是需要战胜的对手。在疼痛中继续高强度，往往把"小信号"拖成"真损伤"：

- 痛感会让你不自觉改变动作 → 代偿 → 新的部位受累；
- 当下靠肾上腺素压住，次日才暴露真实代价；
- 一次"忍痛 PB"换来两周停训，是负收益。

所以规则很硬：**疼痛等级与计划冲突时，以疼痛为准**。

## 一个实用判断:能减轻 vs 会加重

- 热身后减轻、跑中不加重、次日不加重 → 通常可在低强度内谨慎进行；
- 越跑越重、或次日加重 → 降级并复盘，别再叠负荷。

## 适用场景

跑前自检判断能不能跑；跑中疼痛判断要不要停；复盘时给一次疼痛打分以追踪趋势。

## 最小行动

现在给身上最明显的一处不适打分(0-10)，并记下位置、性质、时机三项。

## 费曼讲解问题

- 疼痛分级到底在帮我做什么决策？
- 为什么"次日加重"比"训练中忍住"更重要？
- 哪种疼痛性质要格外警惕？为什么？
- "能减轻 vs 会加重"怎么帮我当场判断？
- 今天我能做的最小动作是什么？
