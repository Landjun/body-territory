# 轻松跑原则

## 本页目的

纠正"慢跑没用"的错觉，明确当前阶段以低强度打底为主。

## 核心原则

- 慢跑不是弱；
- 低强度是打底；
- 当前阶段不要迷恋配速；
- 能说话的强度（说话不费力）；
- 次日不加重；
- 安全完成；
- 跑后能继续工作学习。

## 适用场景

- 恢复期、基础期的大部分跑步；
- 状态一般、想维持身体节奏的日子。

## 怎么判断"够轻松"

- 能完整说出一句话不喘；
- 跑完不瘫，还能继续学习工作；
- 第二天没有明显加重的疲劳或疼痛。

## 最小行动

下次跑步时全程保持"能说话"的强度，跑完记录次日感受。

## 费曼讲解问题

- 轻松跑用一句话怎么讲？
- 为什么"慢"在当前阶段反而正确？
- 它如何帮助我减少受伤？
- 它如何服务网络安全主线？
- 今天我能做的最小动作是什么？
