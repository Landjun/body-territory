# 红旗信号

## 本页目的

列出必须立即停止跑跳、必要时就医的高风险信号。

## 红旗信号清单

- 疼痛 5+；
- 步态改变；
- 疼痛越来越重；
- 次日明显加重；
- 肿胀；
- 麻木；
- 功能受限；
- 跑步中尖锐疼痛；
- 休息后仍不缓解。

## 出现红旗信号怎么办

出现以上任一信号时：**停止跑跳，必要时咨询专业人士。**
不要"再试一下"，不要"忍一忍跑完"。

## 免责声明

本页为个人训练安全提醒，不是医疗诊断或治疗方案。出现持续或加重的疼痛、肿胀、麻木、功能受限等情况，请及时咨询医生、康复师或其他合格专业人士。

## 核心原则

红旗信号面前，没有"完成训练"这个选项，只有"保护身体"。

## 最小行动

把这份清单记牢，跑步中一旦出现任意一条，立即停。

## 费曼讲解问题

- 红旗信号用一句话怎么讲？
- 为什么红旗信号面前不存在"忍一忍"？
- 它如何帮助我减少受伤？
- 它如何服务网络安全主线？
- 今天我能做的最小动作是什么？
