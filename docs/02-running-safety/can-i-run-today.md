# 今天到底能不能跑(每日决策总入口)

> 一句话讲清:这是把跑前 5 问、疼痛分级、睡眠能量、次日反馈,合成一条"五分钟就能走完"的每日决策流——它的输出是一个训练等级 + 一个具体动作,而不是"凭感觉"。

> ⚠️ 个人训练记录,非医疗建议;出现持续/加重疼痛、肿胀、麻木、步态改变等,请咨询医生或合格专业人士。

## 这篇解决什么问题

知识再多,落不到"今天该怎么练"就没用。这一页把所有信号收敛成一个决策,确保**疼痛优先于计划、状态决定强度、最差也有非零选项**。

## 决策流(从上往下走)

```text
① 安全先行:有红旗信号吗?(疼痛5+ / 步态改变 / 肿胀 / 麻木 / 休息不缓解 / 尖锐刺痛)
     是 → 停止跑跳,必要时就医。今天不跑,只做温和活动。结束。
     否 → 继续

② 疼痛 0-10(腿/膝/踝/足/小腿,取最高):
     5+   → 不跑跳(回到①的处理)
     3-4  → 不跑步 → 走 D 级(快走+康复力量+拉伸),并写复盘
     0-2  → 继续

③ 次日反馈:上一次训练后次日加重了吗?
     是 → 降一级 + 复盘诱因(别在未恢复上叠加)
     否 → 继续

④ 睡眠 & 能量:
     睡眠 < 6h → 降一级
     能量 < 4  → 再降一级(可叠加)

⑤ 今日可用时间:
     只有约 10 分钟 → 走 E 级(5 分钟保底,保证不断档)
     20-60 分钟    → 按上面得到的等级执行

⑥ 还是不知道做什么 → 默认:快走 + 基础力量(臀桥/提踵/死虫)+ 拉伸
```

## 输出:等级 → 动作

| 落到的等级 | 今天做什么 |
|---|---|
| A | 完整训练(热身→低强度跑/快走跑→力量→拉伸) |
| B | 基础(快走+慢跑+臀桥/提踵/核心+拉伸) |
| C | 低强度维护(快走+轻力量+拉伸,不跑强度) |
| D | 不跑步(快走+康复力量+拉伸+写复盘) |
| E | [5 分钟保底](../03-strength-rebuild/minimal-5min-routine.md) |

等级定义详见 [A-E 训练等级](training-level-a-e.md);降级路径见 [降级规则](../04-injury-review/downgrade-rules.md)。

## 和安全跑者的关系

这条流就是"安全跑者"每天的核心动作:让**身体信号**而不是计划表或情绪,决定今天怎么练。走一遍只要几分钟,但它是无伤长期主义的发动机。

## 和网络安全纪律的关系

等同于"操作前的风险评估 + 限流决策":先看告警(疼痛/红旗),再看负载(睡眠/能量),决定是全量、降级还是只跑最小心跳包(E 级)。没评估就操作,身体和系统都一样会出事。

## 常见误区

- 跳过①直接看计划——红旗信号必须最先处理;
- 多个降级条件只算一个——它们可叠加,该降几级降几级;
- 把 E 级当"没练"——E 级是保连续性的最小心跳,不是零。

## 今天可以怎么用

现在就走一遍这条流,把结果(等级 + 动作)写进今天的[训练日志](../05-training-log/daily-log-template.md)。
