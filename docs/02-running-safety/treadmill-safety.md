# 跑步机安全

> ⚠️ 个人训练记录,非医疗建议;出现持续/加重疼痛、肿胀、麻木、步态改变等,请咨询医生或合格专业人士。

## 本页目的

把阳台的跑步机当成一个可控的低强度训练工具，而不是硬撑的场所。

## 核心原则

- 跑步机是工具，不是硬撑场所；
- 跑步机适合低强度、可控训练；
- 注意速度、坡度、扶手、空间、安全夹；
- 疼痛时不在跑步机上硬跑；
- 训练结束要记录身体反馈。

## 使用清单

- 开始前：检查安全夹/急停、周围空间、鞋带；
- 起步：从慢走开始，再逐步提速；
- 过程中：保持自然步态，不死盯屏幕、不过度抓扶手；
- 结束：逐渐降速到慢走，再停。

## 适用场景

- 天气、时间不允许外出时的低强度跑/快走；
- 需要精确控制速度和时长的恢复训练。

## 最小行动

下次上跑步机前，先确认安全夹已就位、空间无障碍。
