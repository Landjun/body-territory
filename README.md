# 身体地盘 · Body Territory

> 个人成长三大支柱之第三支柱「身体健康 / 跑步」的体系化学习作品集与训练日志。
> 前两个支柱(传统安全、AI 安全)已有各自的技术作品集,本仓库补齐「身体」这一块。

这是一个用 **费曼学习法** 构建的结构化知识库。核心理念:

> **每个知识点都要能用大白话讲给一个完全不懂的外行听。讲不出来 = 没学会。**

仓库里每一篇笔记本质上都是一份"讲义",不是抄书,而是逼自己把概念嚼碎、重新讲出来。

## 🧠 费曼学习法说明

每篇知识点笔记都遵循同一套结构(见 [templates/feynman-note-template.md](templates/feynman-note-template.md)):

1. **一句话讲清** — 用大白话假设听众完全不懂。这一步讲不出来,就说明还没学会,别急着往下写。
2. **费曼检验问题** — 针对该知识点预设的几个"你真懂了吗"的问题,能答上才算过关。
3. **详细笔记** — 把原理拆开讲透。
4. **类比与图示** — 用一个生活类比把抽象概念落地。
5. **常见误区** — 容易理解错的地方。
6. **关联** — 这个知识点和其他知识点 / 其他支柱怎么连起来。

每个知识点有三种状态:

- 🔲 **未学**
- 📖 **学习中**
- ✅ **能讲清楚**(费曼检验全部能答上)
