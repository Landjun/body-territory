# 训练日志 · Training Log

> 理论之外的实践记录。把每次训练写下来,和知识库里的知识点互相印证——这正是费曼学习法的闭环:学到的东西,要能解释自己身上发生的事。

## 组织方式

- 按 `年/年-月.md` 组织,例如 [2026/2026-06.md](2026/2026-06.md)。
- 每个月一个文件,月内每次训练追加一条记录。
- 单次记录可参考 [../templates/training-log-template.md](../templates/training-log-template.md)。
- 每周可做一次复盘,参考 [../templates/weekly-review-template.md](../templates/weekly-review-template.md)。

## 记录原则

- 不光记数字(距离 / 配速 / 心率),也记**感受**和**反思**。
- 出现任何伤病信号,单独标注,关联到 [跑步损伤原理](../02-injury-free-running/concepts/running-injury-principles.md) 与 [负荷管理](../02-injury-free-running/concepts/load-management-10-percent.md)。

> ⚠️ 个人训练记录,非专业训练建议。
